Introduction

This is a collections of Pure CSS3 Loader Animations.

These 24 CSS3 animation effects are compatible with all major browsers: IE10+, Firefox, Chrome, Safari, Opera…

The collection is very easy to use, you only need to add the style sheet to your page and add a simple html code to display your preferred CSS3 Loading animation!

You have choice between various Loader Effects. You can also change the CSS3 file to customize the effects, like the colors, animation speed…

There is one index.html file of 24 loaders.

Add html code to your page with a selected loader. For example flower-03 loader.

Then add a styles from loader00.css file to your style sheet with a name of a selected loader.
